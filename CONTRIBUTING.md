# Contributing

Thanks for your interest in contributing to pi-laoshi.

## How to contribute

- Open an issue to discuss larger changes before implementing them.
- Keep changes focused and easy to review.
- Include tests when adding behavior once the project has a test setup.
- Update documentation when behavior, commands, or project structure changes.

## Development status

This project is currently in the planning/scaffolding phase. The implementation plan lives in [`docs/plans/pi-laoshi-agent-plan.md`](docs/plans/pi-laoshi-agent-plan.md).
